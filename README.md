# 🌐 TalkSphere

TalkSphere is a modern, real-time communication and language exchange platform built with scalable technologies. It enables seamless messaging, video calls, and global interaction with a beautiful, fully customizable UI.

---

## 🚀 Live Demo

🔗 **Live URL:**
👉 https://talksphere-v61t.onrender.com

---

## 🚀Features

### 🌐 Real-time Messaging
- Instant messaging with typing indicators
- Emoji-based message reactions
- Smooth real-time updates

### 📹 Video Calling
- 1-on-1 and group video calls
- Screen sharing functionality
- Call recording support

### 🔐 Authentication & Security
- JWT-based authentication
- Protected frontend & backend routes
- Secure handling of environment variables

###🌍 Language Exchange Platform
- Designed for global communication
- 32 unique UI themes for personalization
- User-friendly and interactive experience

### ⚡ Modern Tech Stack
- React + Express + MongoDB
- TailwindCSS for clean UI
- TanStack Query for optimized data fetching
- Zustand for global state management

### 🚨 Robust Error Handling
- Frontend error boundaries
- Backend API error handling

###🚀 Free Deployment Ready
Optimized for free cloud deployment platforms

###🎯 Scalable Architecture
Built using scalable real-time services like Stream

## 🧪 Environment Variables Setup
### 📁 Backend (/backend/.env)
```text
PORT=5001
MONGO_URI=your_mongo_uri
STEAM_API_KEY=your_stream_api_key
STEAM_API_SECRET=your_stream_api_secret
JWT_SECRET_KEY=your_jwt_secret
NODE_ENV=development
```

### 📁 Frontend (/frontend/.env)
```text
VITE_STREAM_API_KEY=your_stream_api_key
```

## ⚠️ Important: Never commit your .env files to GitHub.

##🔧 Run the Project Locally

###▶️ Start the Backend

```js
cd backend
npm install
npm run dev
```

###💻 Start the Frontend

```js
cd frontend
npm install
npm run dev
```

✅ The app will now be running locally 🚀
